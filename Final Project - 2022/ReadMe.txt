************************************************************
* 1.  OVERVIEW
************************************************************
This Web Application allows Students to log into their portal
and view their grades for thier respective subjects as well
as their overall average and overall Grade.
This web application also allows Teachers or Lecturers to log
in and create a student profile and populate thier portal 
with their grades.
Students, Administrators and/or Teachers will be able to 
create an account in the web application
************************************************************
* 2.  SYSTEM REQUIREMENTS
************************************************************


1.  The system must be running on one of the following
    operating systems:
    
    - Microsoft Windows Vista or Higher
    - Mac OS
    - Linux
   

2. The following operating systems are not supported:

    Any version of the following Microsoft operating systems:
    - MS-DOS
    - Windows 3.1
    - Windows NT 3.51
    - Windows 95
    - Windows 98
    - Windows NT 4.0
